It is widely used language on web to develop web pages and web applications.

It is developed by Burners Lee in late 1991.

It is a case insensitive language.

It is a tag based language.
ex:
	<html> 
	<HTML>

HTML tags contain opening tag and closing tag.
ex:
	<html>
	</html>

HTML tags may contain attributes.Each attribute contains attribute name and attribute value.
ex:
	<body bgcolor="red">

HTML tags may contain multiple attributes and each attribute must be seperated with space.
ex:
	<body bgcolor="red"  background="image">

In HTML ,we can't create custom tags.
ex:
	<ihub>   //invalid 

HTML document we need to save with .html or .htm extension.

A simple HTML document is also known as component.

HTML is used to develop client side components.

All the HTML documents will execute in browser window.
ex:
	Chrome 
	Mozilla
	Microsoft Edge
	Opera
	IE 
	Chromium
	an etc.

HTML document will be created in below editors.
ex:
	Notepad 
	Notepad++
	Sublime
	VSC 
	Dreamviewer 
	and etc.

HTML2.0 specification was released in 1995.

HTML4.0 specification was released in 1999.

HTML5 which is a extension of HTML4 was release in 2012.

The main objective of HTML5 is used to develop leight weight components.

HTML5 is also known as Advanced Hypertext Markup language.	


HTML document Skeleton
=======================
<!DOCTYPE html>
<html>
	<head>
		// head related tags 	
	</head>
	<body>
		// body related tags 
	</body>
</html>

Note:
------
To represent HTML document we need to use <!DOCTYPE>.
To represent HTML5 document we need to use <!DOCTYPE html>.

<html>
-------
It is a root tag for entire HTML document.

<head>
-------
A <head> tag is used to declare following things.
ex:
	title of a web page
	favicon
	styles
	 
<body>
------
A <body> tag is used to declare actual content of a web page.


ex:
----
<!DOCTYPE html>
<html>
	<head>
		<!--  title -->
		<title>IHUB TALENT</title>

		<!-- favicon -->
		<link rel="icon"  href="insta.png">
	</head>
	<body bgcolor="cyan">
		
		Welcome to HTML classes
	</body>
</html>

HTML Meta tag 
============== 
A meta tag is used to declare metadata of a document.

metadata means data about data.metadata is summarize the data that lead us to details data.

for example text book inside text book index pages is there that is one data this index pages provides the another data means detail information.

Metadata is used by browser window, search engine and other web services.

To declare a metadata we need to use <meta> tag.

A <meta> tag must be used inside <head> tag.

Using <meta> tag we can declare following things.

1) Description

2) Auther

3) Keywords

4) Viewport

5) Refresh

6) Copyright 

and etc.

UTF-8
------
UTF stands for Unicode Transformation Format.

It is a encoding method to describe what character set a website is written with.
"it is a encoding method to describe what character set we have use in our web page".


ex:
